How to submit or select sequences/genomic regions for analysis
 
SynView provides three easy-to-use options for submitting or selecting sequences for an analysis in SynView.  Each option is accessible by clicking the tab in SynView's front-page.  They are:
 
1. CoGe Database Retrieval:  SynView is directly connect to our in-house genomes database, CoGe, and can retrieve genomic sequences (although with their associated genomic features) from it.
2. Internet Sequence Retrieval: If you know the genbank accession for a sequence like to analyze in SynView, simply type it in and SynView will retrieve it directly from NCBI.
3. Direct Sequence Submission:  If you have your sequence in either fasta or genbank format, you can cut-and-paste it into a textbox.
 
Feel free to mix-and-match any of the above sequence submission methods too!
How do I use "CoGe Database Retrieval" to select a genomic region for comparison?
 
The expanded view of this tab is show above.  Here, you will see a series of three columns that allow you to type a name in a text box, a radio button for reverse the display of the region, a text box called "Upstream" and another textbox called "Downstream.  In order to make genomic navigation relatively easy, SynView allows you to select a genomic region by using the name of a feature within your region of interest (e.g. a gene).  In the example above, the gene name "at1g01010" was typed into the "Name" textbox for Sequence 1.  When you have finished typing the gene name, defocus the cursor (e.g. click on some other part of the web-page). When you defocus, SynView will then check in CoGe to see if there are any genomic features by that name.  If there are, it will then return two scroll menus.  The first contains the genomes of organisms that contain that contain a genomic feature by that name and the second contains the genomic locations for all genomic features for the genome selected.  In the example above, you will see that the organism is "Arabidopsis thaliana: chr1.xml (TAIR v6)" and the genomic location is "Chr:1 3631..5899".  If you look closely at the image, you will notice "(2)" after the organism and "(1)" after the location menus.  These are the counts of the number of items in the menus.  The reason there are two genomes with a feature with the name "At1g01010" is because there are two versions of the Arabisopsis genome in CoGe (version 5 and version 6).   If the name cannot be found in CoGe, you will get a message back that says "Accession not found".
 
Once you have entered a genomic feature name that exists in the database, you then can specify the amount of sequence upstream and downstream of that region to use.  In the above example, this has been set to "10000" for both which mean grab 10,000 nucleotides upstream and downstream of the gene.  In all cases, upstream refers to sequences to the "left" (lower number in genomic coordinates) of the gene regardless of whether the gene in on the top or bottom strand; downstream to the right.  If you notice, the gene in the example starts at position 3631 which means there is only 3630 nucleotides upstream from it.  Only the available is used in the analysis.  Also, SynView supports using negative numbers in the upstream and downstream boxes.  This allows you to narrow in a particular sub-sections of a genomic feature if you wish to use it in an analysis without the surrounding sequence.

How do I use the "Internet Sequence Retrieval" option?
 
All you need to do is to type in a GenBank Accession in the textbox and select a start position.  Then, you can also specify how much sequence upstream and downstream from the start position you would like.  With the default settings, this will use the entire sequence as retrieved from GenBank.  When you run your analysis, SynView will then retrieve the sequence from GenBank along with all the associated annotations for use in the analysis.
How do I use the "Direct Sequence Submission" option?
 
Simple!  Just cut and paste your sequence into the textboxes provided.  SynView supports sequences in Fasta or GenBank format.  One advantage to GenBank format is the ability to encode annotations for your sequence which, if provided, will be displayed in the results and can be used to mask coding or non-coding sequences.  There are radio buttons for generating the reverse complement of your sequence as well as start and stop textboxes for specifying a specific range of your sequence.

How do I determine which genomes and genomic features are in CoGe?
 
If you wish to browse genomes in CoGe, please visit GeLo: CoGe's genome location system.  If you wish to browse for genomic features, please visit FeatView: CoGe's genomic feature viewer.
