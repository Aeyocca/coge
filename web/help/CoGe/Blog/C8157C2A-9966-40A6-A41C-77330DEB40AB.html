CoGeX
 
Recently, much work has been done to migrate CoGe's database API from using Class::DBI to DBIx::Class for the base class.  This switch offers major advantages for customizing and prefetching DB queries which should increase the performance of CoGe's system.  We have currently migrated SynView to the new API and have halved the DB access time for the application.  Big thanks to Brent Pedersen for lots of help in the migration and performance tuning, and Brian Thomas for laying the foundation and spear-heading this project.
