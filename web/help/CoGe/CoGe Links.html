Home Page for the CoGe online system.  Provides links to all of CoGe's online tools.  Visit here for more information.
SynView is CoGe's application for comparing up to three genomic regions using bl2seq and our unique genomic graphics package, GeLo, for visualizing results. Visit here for more information.
SeqView is CoGe's application for getting genomic sequence from CoGe's genome database.  It is accessed through other applications in CoGe such as FeatView and GeLoView.  Visit here for more information.
FeatView is CoGe's application for searching all of the features in CoGe's genome database by name, type, and/or organism.  Although help pages have not yet been written for this tool, it is easy to use.  Simply type a feature name and the search begins.
GeLoView is CoGe's application for viewing information or a region of a genome in CoGe's genome database.  Although help pages have not yet been written for this application, it is easy to use.  Just select or search for an organism.  Once selected, you can launch the Genome Viewer or select a region for SeqView.
