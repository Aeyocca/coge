What internet browsers does CoGe's on-line tools support?

CoGe's on-line tools were developed using Firefox. We have conducted limited testing of CoGe's tools using Safari (available for Mac OS X) and Internet Explorer. Safari works for all tested applications (but if you experience problems, we'd like to hear about them), but we have found many problems using Internet Explorer. Thus, we commend that you download and install Firefox. We apologize in advance for the inconvenience this may cause you.
