How do I specify options for bl2seq?
 
Under the “Bl2seq Parameters” tab is where you can set various parameters for bl2seq.  Here you can select which blast program to use and set commonly used options such as gap penalties, word-size, e-value cutoff, etc.  The blastn default values we use for plant genomic comparisons and finding conserved non-coding sequences are given above.  You can also specify additional options in the text box “Other Blast Parameters." If you select to use tblastx, please be more patient (than usual) when waiting for your results as this is performing 6 frame translations on both sequences.  Also, the results from this analysis are filtered to remove HSPs with stop codons that overlap with coding sequence.  This helps cut-down the noise returned from the analysis and makes visualizing the results easier.  However, if you would like to see all the HSPs, you can always view and download the raw blast report for further analysis from the SynView results section.  Although we try to keep the documentation as up to date as possible, extending the functionality of the types of blast analyses is an active area of development in SynView.  Thus, if this area of SynView is different that what is shown here, don’t worry!  The version you are using is more recent and hopefully more useful as well.
